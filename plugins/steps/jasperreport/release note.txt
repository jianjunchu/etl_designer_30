kettle jasper report.
this plugin can generate a jasper report(PDF/EXCEL/HTML format) by binding jrxml file with data source generated by kettle.


1. environment
windows 2000/XP/2003 
linux
JVM 1.5 or above
Kettle 3.1 or above

this plugin need a higher version of jfreechart.jar 
You can replace the old jfreechart-1.0.1.jar (in ./libext/jfree directory) with a new one( jfreechart-1.0.12.jar is better).
